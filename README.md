# Infinite-Int
A custom number class that supports arbitrarily large integers without using Javaâ€™s built-in BigInteger.
Numbers are stored as a linked structure and formatted with commas.

## Features
- Accepts large integer inputs as strings
- Validates input and rejects illegal characters
- Supports:
- Addition
- Comparison
- Reversing/printing digits
- Custom comma-formatted toString() output
- Handles very large numbers that exceed standard integer limits
  
## How to Run
This program runs on:
- OnlineGDB
- Replit
- Any local Java compiler/IDE
